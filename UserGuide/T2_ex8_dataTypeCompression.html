% Start IP Explorer
web("https://ipexplorer.arm.com/");
You can prepare software applications for benchmarking on Arm processors and run them with IP Explorer.
You need an Arm account to access IP Explorer. If you need a new account visit Arm Developer and click the person icon in the top right corner and click Register. If you don't have access to IP Explorer you will get a form to request access. Fill out the form and somebody will review and grant access.
Choose one IP of interests.
Set up the IP Simulation configuration.
Choose preloaded example.
The simulation will start and the detailed steps can be monitored in the console tab.
You can find the code compiled and executed from the downloaded examples.
