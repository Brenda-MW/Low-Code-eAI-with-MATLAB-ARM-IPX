Generate Simulink Models for Simulation

Export trained model into simulink

Export the top features from the Classification Learner app using Export Model > Export Model to Simulink.
This will save a trainedModel as SLX file to open.
We will use a saved model to open and simulink.
modelName = "KNNModelFromClassificationLearner.slx";
open_system(modelName);
sim(modelName);
